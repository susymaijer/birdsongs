When the installation of "Birds of Venezuela" fails

Sometimes the setup program  fails, for unclear reasons. Fortunately there is an alternative way to install the program:

Four steps:

  1. Create a directory ("folder") with a name with at most 8 characters (for example: "venbirds").
  2. Download this zip-file, unzip it and put the resulting files in the newly created directory.
  3. Double-click on the program file VENBIRDS.EXE (one of the files in the zip-file) to start the program.
  4. When you try to play a sound, the program will ask you for the drive letter of the CD-ROM drive. After you enter it, program setup is complete.
