Program Bugs of the CD-ROM "Birds of Tropical Asia 2.0"

by Sjoerd Mayer

  • Loading sound files into a sound editing program (eg.. Cool Edit) from a copy of the CD-ROM on your hard disk, is not possible if the name of the directory in which the copy resides contains a space. This problem is solved if you run the patch program (click here to read more).
