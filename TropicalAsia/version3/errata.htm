Patch Program and Errata of "Birds of Tropical Asia 3"

Patch Program (current version: 3.15)

Click here to run a program which "patches" the installed program of "Birds of Tropical Asia 3" so that:
- the erroneous sound recordings are either no longer visible, or are moved to the correct species, and
- the program bugs (see below) are fixed, and
- the program can now make mp3 files from all recordings (except one: rec. #3 of Long-tailed Bush-Warbler Bradypterus caudatus). Do this with File/Make All MP3 Files, with the popup menu on the soundbars, or with "Extra/prepare for mp3 player" in the Playlists window. The mp3 files have all the recording information embedded (species, sound type, locality, etc)!

Errata in version 3.0

  • The recordings of the Temminck's Tragopan Tragopan temminckii are not of this species, but most likely the song and calls of the Red-winged Laughingthrush Garrulax formosus. Discovered by Paul Holt. June 2006. Corrected on version 3.5.
  • The recording of the Verreaux's Monal-Partridge Tetraophasis obscurus is not the call of this species, but the call of the White Eared-Pheasant Crossoptilon crossoptilon. Discovered by Paul Holt. June 2006. Corrected on version 3.5.
  • The first recording ("Contact calls") of the Chestnut-headed Tesia Tesia castaneocoronata is not the call of this species, but the call of the Grey-bellied Tesia Tesia cyaniventer. Discovered by Paul Holt. June 2006. Corrected on version 3.5.
  • The recording of the Rufous-vented Laughingthrush Garrulax gularis is not the chorus of that species, but probably the chorus of White-crested Laughingthrush Garrulax leucolophus. Discovered by Paul Holt. June 2006. Corrected on version 3.5.
  • The recording of the Sambar Cervus unicolor is not the call of this species, but the call of a Barking Deer Muntiacus muntjak.
    Pointed out on 20 September 2005 by Erik Meijaard, and corrected on version 3.3.
  • send e-mail to Jelle Scharringa if you found an error not listed above.

Program Bugs in earlier version(s) 3.x

  • Error with "Extra/Prepare for Audio CD..." in the Playlist window.
    The program could not convert a number of soundfiles.
  • Error showing two photos.
    The program was unable to show the photos of Siberian Thrush Zoothera sibirica and Middendorf's Warbler Locustella ochotensis.
  • A problem with very long playlists was found and fixed.
  • A message about an "outdated scrambling method" of photos no longer pops up.
  • In version 3.8, MP3 files were written to folder "...\Birds of Tropical Asia 1" instead of "...\Birds of Tropical Asia 3".
  • On Windows Vista, version 3.9 (sometimes?) couldn't play/show sounds and photos when they were copied to the default folder with Options/Copy to Hard Disk.
  • send e-mail to Sjoerd Mayer if you think you found a bug not listed above.
